# SDKWork Course Mini Program Application

This is the mini program application root for the SDKWork course management system.

## Architecture

- **Root**: `apps/sdkwork-course-mini-program/`
- **Standard**: `MINI_PROGRAM_APP_ARCHITECTURE_SPEC.md`
- **Platform**: WeChat Mini Program

## Package Taxonomy

- `sdkwork-course-mp-core` — Core runtime, SDK clients
- `sdkwork-course-mp-commons` — Shared components
- `sdkwork-course-mp-courses` — Course browsing
- `sdkwork-course-mp-lessons` — Video learning
- `sdkwork-course-mp-live` — Live sessions

## Development

```bash
# Install dependencies
pnpm install

# Build for mini program
pnpm build
```
